‚öõÔ∏è What is React Native

  • Built using React
  • Build cross patform apps using JavaScript
  • Target not only iOS & Android, but also Apple TV, VR, AR, Windows, & Desktop
  • Real world projects typically see 80%-95% code reuse

Why React Native

  • Fast development speed
  • Low development cost
    • Code reuse
    • Easier to find developpers
  • Ship across multiple platforms
  • Over-the-Air updates

How React Native works

  • JavaScript is bundled and minified using Babel (built-in)
  • Separate threads for UI, Layout, and JavaScript
  • Threads communicate asynchronously through native bridge
